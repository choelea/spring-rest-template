[[overview_http_verbs]]
== HTTP请求动词

在使用HTTP动词时会尽量遵循标准的HTTP和REST约定。

|===
| 动词 | 用处

| `GET`
| 用来获取资源

| `POST`
| 用来创建新的资源

| `PUT`
| 用于更新已有资源

| `PATCH`
| 用于更新现有资源，包括部分更新

| `DELETE`
| 用于删除已有资源
|===

[[overview_http_status_codes]]
== HTTP 状态码

在使用HTTP状态代码时尝试尽可能地遵守标准HTTP和REST约定。

|===
| Status code | Usage

| `200 OK`
| 请求成功完成

| `201 Created`
| 已成功创建新资源
`Location` header

| `204 No Content`
| 对现有资源的更新已成功应用

| `400 Bad Request`
| 该请求格式不正确。响应主体将包含一个错误，以提供更多信息

| `404 Not Found`
| 该请求格式不正确。响应主体将包含一个错误，以提供更多信息
|===
